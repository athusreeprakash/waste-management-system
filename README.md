# waste-management-system
A Django-based Waste Management System with waste pickup, user &amp; collector modules, and points-based rewards.

## Features
- User registration and login
- Waste collector registration
- Waste pickup request and tracking
- Category-based waste management
- Points and rewards system
- Complaints management
- Product purchase using points
- Location (pincode) based services

## Technologies Used
- Python (Django)
- MySQL / MariaDB
- HTML, CSS, JavaScript
